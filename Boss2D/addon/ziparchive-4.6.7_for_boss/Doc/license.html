Introduction

The ZipArchive Library is distributed in two versions: the Open Source (GNU GPL) Version and the Full Version. The Full Version requires purchasing the commercial license and the Open Source (GNU GPL) version may require it. Having the commercial license of the ZipArchive Library has the following benefits in both cases:
  • The commercial license allows you to embed the ZipArchive Library in proprietary software without affecting the copyright of this software
  • The license gives you royalty free distribution rights.
  • The license is granted per developer, team or site.
  • The upgrades of the ZipArchive Library are free.

Open Source (GNU GPL) Version

The Open Source (GNU GPL) Version is licensed under the GNU General Public License. Distributing the ZipArchive Library under GNU GPL implies, that the ZipArchive Library can only be used by software that is licensed under conditions compliant with the GNU GPL. One of these conditions is an obligation to make the source code of the software publicly available. If you find this condition not suitable for your software, you can obtain the commercial license for using the ZipArchive Library that removes this obligation.
